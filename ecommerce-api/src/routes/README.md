# README — src/routes

Rutas agrupadas por recurso. `index.js` monta todas bajo `/api`:
- `authRoutes.js` → `/auth/register`, `/auth/login`
- `productRoutes.js` → `/products`
- `categoryRoutes.js` → `/categories`
- `cartRoutes.js` → `/cart`
- `orderRoutes.js` → `/orders`
- `userRoutes.js` → `/users`
